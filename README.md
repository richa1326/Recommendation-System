# Recommendation System in Python

This project implements a recommendation system using collaborative filtering from scratch.

## Features
- User-based collaborative filtering
- Cosine similarity
- Dataset-driven recommendations

## Dataset
A sample CSV dataset of user-item ratings is used.

## How to Run

python recommendation_system.py

## Output
Recommends items for a given user based on similar users.

## Tech Used
- Python
- CSV processing
- Math (cosine similarity)
